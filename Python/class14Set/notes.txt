Set Data Structure:
    Group of elements as one enitity
    Duplicates are not Allowed but Allowed Heterogenious
    Indexing concept is not Applicable
    Order not Maintained

Set methods:
    s.add(element)
    s.update(seq1,seq2)
    s.copy()
    s.clear()
    s.pop()
    s.remove(element)    #if the element is there it removes or not means it shows error : Key Error
    s.discard(element)   #if the element is there it removes. if not means it won't show any error.


    s1.union(s2)   ---------------------Returns all your elements.
    s1.intersection(s2)
    s1.difference(s2)
    s1.symmetric_difference(s2)
